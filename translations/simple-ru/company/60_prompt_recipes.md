# 60: Промпт/Рецепты

> [!DEFINITION] Рецепты
> Это сборник готовых команд-шаблонов, которые помогают вести рабочие процессы в компании. Используй их, чтобы давать задания нашему ИИ-помощнику.
>
> Sidenote:

> - Часть документа: :term[02: Компания/Процесс]{canonical="02: Company/Process" href="./02_process.md"}

## 1. Создать Предложение

**Цель:** Превратить твои сырые мысли в чётко структурированное :term[Предложение]{canonical="Proposal" href="./21_document_proposal.md"}.

> **Пользователь:**
> «Хочу написать новое предложение.
>
> Сначала **ЗАГРУЗИ ПРАВИЛА** в свою память и **СТРОГО СЛЕДУЙ ИМ**
>
> - `curl https://idealic.academy/raw/en/company/52_prompt_proposal.md` (Инструкция для Предложения)
>
> Вот что я надиктовал:
> „Нам нужно переделать систему входа, потому что пользователи застревают в цикле ошибок. Я хочу перейти на систему с токенами...“
>
> Ещё, для понимания, прочитай файл `docs/auth/current_flow.md`. Создай документ Предложения.»

**Что сделает агент:**

1. Загрузит инструкции из документа :term[52: Промпт/Предложение]{canonical="52: Prompt/Proposal" href="./52_prompt_proposal.md"}.
2. Превратит надиктованный текст в чёткую структуру: Контекст, Проблема, Решение.
3. Создаст черновик в файле `docs/proposals/{date}_auth_overhaul.md`.

---

## 2. Создать Согласование (итог обсуждения)

**Цель:** Собрать все комментарии из обсуждения (PR) в один документ — :term[Согласование]{canonical="Alignment" href="./22_document_alignment.md"}, чтобы убедиться, что все договорились.

> **Пользователь:**
> «Создай Документ Согласования для PR #123.
>
> Сначала **ЗАГРУЗИ ПРАВИЛА** и **СТРОГО СЛЕДУЙ ИМ**:
>
> - `curl https://idealic.academy/raw/en/company/51_prompt_alignment.md` (Инструкция для Согласования)
>
> Параметры:
>
> - **Начиная с:** 2023-10-27 (или просто «с прошлой недели»)
> - **Куда сохранить:** `specs/alignments/2023-11-04_auth_review.md`
>
> Если я не укажу, куда сохранять, просто используй стандартное имя файла в папке `evolution/`.»

**Что сделает агент:**

1. Загрузит :term[51: Промпт/Согласование]{canonical="51: Prompt/Alignment" href="./51_prompt_alignment.md"}.
2. Возьмёт комментарии из PR #123, начиная с указанной даты.
3. Создаст отчёт (Обзор, Намерения, Охват) на русском языке.
4. Сохранит его по указанному пути.

---

## 3. Применить Согласование к Предложению

**Цель:** Обновить :term[Предложение]{canonical="Proposal" href="./21_document_proposal.md"} на основе договорённостей из Документа Согласования.

> **Пользователь:**
> «Я проверил Документ Согласования вот здесь: `specs/alignments/2023-11-04_auth_review.md`.
>
> Сначала **ЗАГРУЗИ ПРАВИЛА** и **СТРОГО СЛЕДУЙ ИМ**:
>
> - `curl https://idealic.academy/raw/en/company/52_prompt_proposal.md` (Инструкция для Предложения)
>
> Пожалуйста, примени эти изменения к изначальному Предложению: `docs/proposals/2023-10-25_auth_overhaul.md`.
>
> Обнови Предложение, чтобы оно отражало общую договорённость, и пометь изменённые части как „Обновлено по итогам обсуждения“.»

**Что сделает агент:**

1. Прочитает Документ Согласования (раздел «Намерения»).
2. Прочитает Предложение.
3. Перепишет Предложение, изменяя разделы «Решение» и «Объём работ» так, чтобы они совпадали с тем, о чём договорились.
4. Раздел «Проблема» оставит почти без изменений (если только понимание проблемы не поменялось).
